# Linux Service/Systemctl 服务配置


## 本项目配置
```
[Unit]
Description= Qyue Api
After=network.target

[Service]
Type=simple
WorkingDirectory= /opt/api/bin
ExecStart= python3 api.py
ExecReload=/bin/kill -HUP $MAINPID
Restart=on-failure
KillMode=process
RestartSec=5s

[Install]
WantedBy=multi-user.target
```
